OverReact Analyzer Plugin

Overview

  • View On GitHub

To make authoring components with OverReact even easier, we provide a Dart analyzer plugin with lints to help spot (and automatically fix) common pitfalls, and assists to make common authoring tasks quicker and easier.

Getting Started

Assuming you have already declared the over_react package as a dependency within your project's pubspec.yaml file, you simply need to enable the plugin by adding the following to your analysis_options.yaml file:

analyzer:
  plugins:
    over_react

Looking for more information about the plugin? Check out the Readme!
