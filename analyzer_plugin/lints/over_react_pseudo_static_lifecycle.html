These ReactJS defaultProps getter and getDerivedStateFromProps lifecycle method are considered static.

Accessing instance methods from within these is prohibited, and will cause runtime errors. The only exceptions are the following "whitelisted" instance members:

newProps
newState
typedPropsFactory
typedPropsFactoryJs
typedStateFactory
typedStateFactoryJs
