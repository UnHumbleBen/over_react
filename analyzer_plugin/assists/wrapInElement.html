wrapInElement

Maturity: stable

Since 1.0.0

View the Project on GitHub workiva/over_react

When a VDom element is selected by the user, the assist wraps the selection with another ReactElement:

EXAMPLE:

final foo = Dom.div()(
  'Hi there foo',
);

BECOMES:

final foo = component()(
  Dom.div()(
    'Hi there foo',
  ),
);
